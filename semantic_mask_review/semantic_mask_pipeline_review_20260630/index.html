Pipeline

1. Task text

Use the episode instruction as the semantic source.

2. LLM object spec

Extract compact object and receptacle prompts; filter robot body words.

3. GroundingDINO

Run prompted proposal detection separately on the three views.

4. Cross-view match

Rank proposal crops with SigLIP2 and DINOv2 embeddings.

5. Local latent mask

Shrink matched boxes into bounded regions on the 40 x 24 latent view grid. Only SigLIP2 and DINOv2 matcher masks are shown.
